🚀 Spring Boot CRUD REST API

A simple CRUD (Create, Read, Update, Delete) REST API built using Spring Boot, Spring Data JPA, and MySQL. This project demonstrates how to create RESTful web services and connect them with a relational database.

📖 Overview

This project provides APIs to manage student records. It covers the complete backend development workflow, including:

* Creating REST APIs
* Connecting to MySQL Database
* Using Spring Data JPA
* Performing CRUD Operations
* Testing APIs with Postman

⸻

🛠️ Tech Stack

* Java 17
* Spring Boot
* Spring Data JPA
* MySQL
* Maven
* Postman
